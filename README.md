The testing software for learning musical notes.

## System requirements
1. Oracle Java >= 15
2. Maven >= 3.8.3

## How to Run?
1. Clone the repo
2. Build the project: execute ```mvn clean install package``` in the project's root.
2. Run it: execute ```mvn jetty:run``` in the project's root.
